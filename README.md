# 📚 API Biblioteca

API REST desenvolvida com o objetivo de praticar conceitos fundamentais do ecossistema Spring Boot e do desenvolvimento Back-end com Java.

## 🚀 Tecnologias Utilizadas

* Java 21
* Spring Boot
* Spring Data JPA
* PostgreSQL
* Maven
* API REST
* Git e GitHub

## 🎯 Objetivos do Projeto

Este projeto foi criado para consolidar conhecimentos em:

* Criação de APIs REST
* Operações CRUD (Create, Read, Update e Delete)
* Persistência de dados com JPA/Hibernate
* Integração com banco de dados PostgreSQL
* Organização de projetos em camadas
* Controle de versão com Git

## 📂 Estrutura do Projeto

```text
controller  -> Recebe as requisições HTTP
service     -> Contém as regras de negócio
repository  -> Responsável pelo acesso aos dados
model       -> Representa as entidades do sistema
```

## 📖 Funcionalidades

* Cadastrar livros
* Listar todos os livros
* Buscar livro por ID
* Atualizar informações de um livro
* Remover livros cadastrados
